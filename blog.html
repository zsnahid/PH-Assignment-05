Important FAQ

What is Document Object Model (DOM)?

The DOM represents a web document with a logical tree. Each branch of the tree ends in a node and each node contains objects. DOM method allows programmatic access to the tree. As an object-oriented representation of the web page, the document's structure, style or content can be modified with a scripting language such as JavaScript.

How do you select an element from the DOM?

In order to be able to manipulate an element in the DOM tree we first have to select that particular element. There are several ways to do this:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
What is event delegation in the context of the DOM, and why is it useful?

Event delegation lets us append a single event listener to a parent element that adds it to all of its present and future descendants that match a selector.

Without event delegation each new child element will need their own event listener. Event delegation ensures that the parent listener will still catch events from new child elements making the code simpler, reducing memory usage and improving code maintenance.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes first we need to select the element using one of the method for selecting elements. Then using the style property we can change the styles of that element. The following example demonstrates that:
HTML: <h1 id="main-heading">This is a headline</h1>
Manipulating the element in the DOM -
const title = document.getElementById('main-heading');
title.style.color = 'lightgreen';
